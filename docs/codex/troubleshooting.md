# Troubleshooting

障害対応、調査結果、暫定対応、恒久対応を記録します。

## 記録ルール

- 障害や調査を行った場合は、発生日、事象、影響範囲、原因、対応、再発防止を記録する。
- 秘密情報、API キー、トークン、Cookie、ライセンスキーを記載しない。
- ログを貼る場合は、必要最小限にして秘密情報をマスクする。

## 記録テンプレート

```md
## YYYY-MM-DD: タイトル

- 事象:
- 影響範囲:
- 原因:
- 暫定対応:
- 恒久対応:
- 再発防止:
- 関連リンク:
```
