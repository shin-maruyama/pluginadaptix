## バグ履歴

### No.1 2026-27-06 FIELD-CHECK-001 設定済みフィールドが削除・変更された場合の警告表示改善

| 項目 | 内容 |
|---|---|
| 重大度 | Medium |
| 分類 | フィールド存在チェック |
| 不具合内容 | 警告は表示されるが、設定番号、保存フィールド名の扱い、現在状態などの情報が不足していた |
| 影響 | 管理者がどの設定を直すべきか特定しづらい |
| 推奨対応 | レコード一覧表示時に毎回警告し、設定項目ごとに対象フィールド情報を表示する |
| ステータス | 修正済み |
