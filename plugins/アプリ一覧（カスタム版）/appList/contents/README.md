# kintone App List Custom Plugin

kintoneアプリの一覧を、ボタン型・ツリー型・ポータル型で表示するカスタムプラグインです。

現在の最新版は `v1.0.27` です。

## 概要

このプラグインでは、kintoneアプリ内に「アプリ一覧」カスタムビューを作成し、設定したカテゴリーごとにアプリへの導線を表示できます。

ポータル型では、カテゴリーカード、番号バッジ、説明文、アイコン付きアプリリンクを使って、社内ポータルのような見た目でアプリを整理できます。

## 実装済み機能

- 表示形式の選択
  - ボタン型
  - ツリー型
  - ポータル型
- デザインテーマ選択
  - `modern`
  - `simple`
  - `classic`
  - `iconic`
- 全体枠デザイン選択
  - `card`
  - `shadow`
  - `leftBorder`
  - `outline`
  - `none`
- 全体ボタンデザイン選択
  - `outline`
  - `solid`
  - `soft`
  - `pill`
  - `square`
  - `underline`
- カテゴリー別設定
  - カテゴリー名
  - サブカテゴリー指定
  - カテゴリー説明文
  - カテゴリー色
  - カテゴリーアイコン
  - カテゴリー別枠デザイン
  - カテゴリー別ボタンデザイン
- アプリ別設定
  - 表示するアプリの選択
  - アプリアイコンの選択
  - 並び順変更
- ポータル型設定
  - PC列数: `4列` / `5列` / `6列`
  - モバイル列数: `1列` / `2列`
  - 説明文行数: `1行` から `5行`
  - カテゴリー名フォントサイズ
  - 説明文フォントサイズ
  - アプリ名フォントサイズ
  - サブカテゴリー名フォントサイズ
- 設定画面内のポータル型プレビュー
- PC版・モバイル版対応

## 設定手順

1. kintoneのシステム管理でプラグインzipを読み込みます。
2. 対象アプリのプラグイン設定画面で、このプラグインを追加します。
3. 表示形式を選択します。
4. カテゴリーを作成し、表示するアプリを選択します。
5. 必要に応じて色、アイコン、枠、ボタン、フォントサイズ、列数を調整します。
6. 保存後、アプリ設定を更新します。
7. 作成された「アプリ一覧」ビューで表示を確認します。

## ポータル型の推奨設定

PCで横幅が十分にある場合は、`5列` または `6列` が一覧性を出しやすいです。

スマートフォンでは、文字の読みやすさを優先する場合は `1列`、一覧性を優先する場合は `2列` を選択してください。

説明文は長さによってカード内の余白に影響するため、通常は `3行` 程度がおすすめです。

## 再パッケージ手順

ソース変更後は、`manifest.json` の `version` と `description` を更新してからパッケージ化します。

```powershell
& 'C:\Users\ALLIN-ONE002\AppData\Roaming\npm\kintone-plugin-packer.cmd' --out 'dist\applist-custom-v1.0.28.zip' --ppk 'dist\glphlnggeoeleppkieoobcllabnjaabe.ppk' .
```

構文確認:

```powershell
node --check js/config.js
node --check js/desktop.js
node --check js/mobile.js
```

ハッシュ確認:

```powershell
Get-FileHash .\dist\applist-custom-v1.0.28.zip -Algorithm SHA256
```

## 注意事項

- プラグイン設定を保存すると、対象アプリに「アプリ一覧」カスタムビューを作成・更新します。
- kintone側でアプリ設定を反映するまで、ビューの変更は本番反映されません。
- モバイル列数は、モバイル表示および狭い画面幅での表示に反映されます。
- 既存のプラグインIDを維持するため、再パッケージ時は既存の `.ppk` を使用してください。
